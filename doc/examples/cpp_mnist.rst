**************************************************
Online inference in C++ with a PyTorch CNN and MPI
**************************************************


This example uses the SILC Fortran client to provide a C++ example of a PyTorch convolutional neural net and
pre-processing script loaded into the Redis database. Random data is then generated by C++ and sent to the
database for online inference.
